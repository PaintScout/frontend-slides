# Sample Prompt — USS Enterprise-D Technical Specifications

## What a user would type

> Build me a fairly in-depth technical presentation about the USS Enterprise
> NCC-1701-D — the Galaxy-class starship from Star Trek: The Next Generation.
> I want real engineering depth: dimensions and mass, the warp propulsion
> system, the computer core, deck/section layout, crew complement, defensive
> systems (shields, phasers, torpedoes), and the saucer-separation capability.
> Audience is Trek fans and engineering-minded folks, so don't dumb it down —
> use the in-universe spec numbers. Make it feel like a Starfleet technical
> readout, not a generic slide deck.

## Generation notes (for fine-tuning comparison)

- **Mode:** A — New Presentation, from scratch (no images, CSS-only visuals).
- **Length:** Medium/Long — target 13–16 slides.
- **Style chosen:** LCARS-inspired ("Library Computer Access/Retrieval System") —
  the curved Okudagram Starfleet interface. Black background, rounded "elbow"
  bars, the canonical LCARS palette (amber `#FF9900`, orange `#FF7700`,
  lavender `#CC99CC`, blue `#9999FF`, salmon). This is deliberately NOT one of
  the 12 presets — it's a context-specific aesthetic, per the skill's
  anti-AI-slop / "designed for the context" rule.
- **Typography:** A condensed, slightly futuristic display face (e.g. Antonio /
  Oswald-style) for the LCARS feel; clean mono or grotesque for spec readouts.
- **Content depth:** Use the well-established in-universe figures — 642.5 m
  length, 42 decks, ~1,014 crew, warp 9.6 max, etc. Lots of labeled stat
  readouts and data panels.
- **Inline editing:** No (keep the file clean).
